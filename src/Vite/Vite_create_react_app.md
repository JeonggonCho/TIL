## 1. 비교

|           | Vite  | CRA(create-react-app) |
|-----------|-------|-----------------------|
| 설치 소요 시간  | 약 30초 | 약 1분 40초              |
| 파일 크기     | 45MB  | 221MB                 |
| 빌드된 파일 크기 | 150KB | 553KB                 |

- Vite가 CRA에 비해 매우 속도가 빠른 것을 확인 할 수 있음
- 하지만, 무조건 Vite가 좋다, 사용해야한다는 의미가 아니며, Vite가 신기술인만큼 안정성에 대한 검증이 필요함
