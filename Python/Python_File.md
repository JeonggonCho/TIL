## (1) 파일 입력

### - open(file, mode='r', encoding=None)

- file : 파일명
- mode : 텍스트 모드
- encoding : 인코딩 방식(일반적으로 utf-8 활용)

| mode | 의미          |
|------|-------------|
| 'r'  | 읽기로 열기(기본값) |
| 'w'  |             |
| 'x'  |             |
| 'a'  |             |
| 'b'  |             |
| 't'  |             |
| '+'  |             |
